---
title: "Test Bank Accounts"
---

When running in non-production environments, you can use specific BSB codes to simulate different results when verifying bank details.



| BSB Code | Simulated Actual Bank Account Name | Simulated Result |
|----------|--------------------------|------------------|
| `111111` | N/A | **No account is found** for the provided BSB and account number. |
| `222222` | N/A | The account matching the provided BSB and account number is **closed**. |
| `333333` | `Random name` | The actual bank account name **does not match** the provided account name. |
| `444444` | `{Provided name} AND SONS` | The actual bank account name **is a close match** to the provided account name. |
| Any other BSB | `{Provided name}` | The actual bank account name **matches** the provided account name exactly. |
